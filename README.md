## Core Primitives
- Widgets: The visual elements.
- QMainWindow: The application frame.
- QWidget: A generic container (like a <div> in HTML).
- QPushButton, QTextEdit, QLabel: Self-explanatory.
- Layouts: PyQt does not use CSS for positioning. It uses "Box Layouts" (like Flexbox).
- QVBoxLayout: Stacks items vertically.
- QHBoxLayout: Stacks items horizontally.
- QListWidget: A list of items.
- QListWidget.itemClicked.connect(self.my_function): Connects the item click event to a function.
- Signals & Slots: The wiring.
    button.clicked.connect(self.my_function)
